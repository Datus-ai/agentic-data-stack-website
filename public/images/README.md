# Social Media Images TODO

## Required Images for SEO

### 1. Open Graph Image
**File**: `og-image.png`  
**Dimensions**: 1200 × 630 px  
**Format**: PNG or JPEG  
**Purpose**: Facebook, LinkedIn, Discord previews

**Content suggestions**:
- Agentic Data Stack logo/branding
- Tagline: "How Data Systems Evolve for Agents"
- Clean, professional design
- Dark theme to match website

### 2. Twitter Card Image
**File**: `twitter-card.png`  
**Dimensions**: 1200 × 630 px (same as OG)  
**Format**: PNG or JPEG  
**Purpose**: Twitter/X previews

**Content suggestions**:
- Can be same as og-image.png
- Or Twitter-optimized variant
- Include @AgenticDataStack handle

### Design Tools Options

1. **Canva** (easiest):
   - Use "Open Graph" template (1200×630)
   - Upload logo
   - Add text and styling
   - Export as PNG

2. **Figma** (professional):
   - Create frame: 1200×630
   - Design with brand colors
   - Export 2x for retina displays

3. **AI Generation** (quick):
   - Use Midjourney/DALL-E with prompt
   - "Professional tech community banner, data architecture theme, dark mode, 1200x630"

4. **Screenshot + Edit**:
   - Take screenshot of website hero section
   - Crop to 1200×630
   - Add overlay/branding

## Brand Guidelines

**Colors** (from website):
- Primary: Violet gradient (#7C3AED to #3B82F6)
- Background: Dark (#0F172A, #07111F)
- Text: Light (#F8FAFC, #E2E8F0)

**Fonts**:
- Inter (or Geist Sans from website)
- Bold for headlines
- Medium for body text

## After Creating Images

1. Save files to `/public/images/`
2. Verify dimensions: `file og-image.png`
3. Optimize: Use TinyPNG or ImageOptim
4. Test previews:
   - Facebook: https://developers.facebook.com/tools/debug/
   - Twitter: https://cards-dev.twitter.com/validator
   - LinkedIn: Share a post and check preview
